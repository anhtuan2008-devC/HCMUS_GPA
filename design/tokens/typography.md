# HCMUS GPA Typography Tokens

## Font Stack

- UI/body: `"Segoe UI", Roboto, Arial, sans-serif`
- Display/headline: `"Segoe UI", Roboto, Arial, sans-serif`

Theo trang nhận diện HCMUS, chữ đi kèm logo dùng Segoe UI Bold và Segoe UI Regular. App dùng cùng hướng sans-serif để gần nhận diện trường, tránh cảm giác editorial lạ với Fraunces.

## Roles

- `heading-xl`: 48-64px, weight 700, line-height 1.05
- `heading-lg`: 32-40px, weight 700, line-height 1.15
- `heading-md`: 22-28px, weight 700, line-height 1.2
- `body`: 15-16px, weight 400-500, line-height 1.65
- `caption`: 12-13px, weight 500-600, line-height 1.45

Ưu tiên độ rõ, dễ quét số liệu và nhãn học tập hơn hiệu ứng chữ trang trí.
